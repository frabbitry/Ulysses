The Ulysses Project

The Ulysses Project is a project in the works that will attempt to do the following:

  • Use XML to tag Ulysses by James Joyce for literary allusions, other references, and locations.
  • Use that marked up document to compile a map that will show the Dublin locations where parts of the book are set.
  • Correlate the locations where the book are set with the types of references and allusions made at those locations.

This project is being done through the Digital Humanities: Coding and Visualization class at the University of Pittsburgh at Greensburg.

The project team members are Fiona, Kiara, and Brad.
